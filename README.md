# API Mahasiswa - Flutter & Laravel

Project ini merupakan implementasi REST API menggunakan Laravel dan integrasi aplikasi mobile menggunakan Flutter. Sistem ini digunakan untuk mengelola data mahasiswa dengan fitur CRUD (Create, Read, Update, Delete).

## Fitur
- REST API Laravel
- CRUD Data Mahasiswa
- Testing API menggunakan Postman
- Dokumentasi API menggunakan Swagger
- Integrasi Flutter dengan API
- Menampilkan data mahasiswa pada aplikasi mobile

## Teknologi yang Digunakan
- Laravel
- Flutter
- Dart
- MySQL
- Postman
- Swagger / L5-Swagger

## Endpoint API
- GET /api/mahasiswa
- POST /api/mahasiswa
- PUT /api/mahasiswa/{id}
- DELETE /api/mahasiswa/{id}
